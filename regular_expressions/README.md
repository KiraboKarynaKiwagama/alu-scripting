# Regular Expressions (Regex)

A collection of scripts where I practice using regular expressions for the first time.

## What is Regex?
A regular expression (regex) is a sequence of characters that defines a search pattern.
It is used to match, find, and manipulate strings in text.

## Files
| File | Description |
|---|---|
| `file1.py` | Matching phone numbers and reformatting them |
| `file2.py` | Validating email addresses |

## Concepts Practiced
- Character classes `[a-zA-Z0-9]`
- Anchors `^` and `$`
- Quantifiers `+`, `*`, `{n,}`
- Groups and backreferences `(\d\d\d)` and `\1`
- Dot wildcard `.`
- Escaping characters `\.`
